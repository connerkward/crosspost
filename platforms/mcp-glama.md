# Glama (MCP Server Registry)

https://glama.ai/mcp/servers

## Submission method
- Submit via GitHub URL. Glama indexes MCP servers from GitHub repos.
- Submit page: https://glama.ai/mcp/servers/submit
- May also auto-discover from GitHub topics (`mcp-server`)

## How to submit
1. Ensure the repo has a proper README with MCP server description, tools list, and setup instructions.
2. Navigate to https://glama.ai/mcp/servers/submit and submit the GitHub repo URL.
3. This likely requires browser interaction or their API if one exists.
